# Kamvim 

This is my vim setup. Have fun with it. 

## Plugins
	* NERDtree
	* NERDcomment
	* surround
	* tagbar
	* tabular
	* Fugitive
	* Command-T

## Install
    git clone git://github.com/Kamii/kamvim.git
Then move .vimrc to home/<user>

To install Command-T you need to run this:

    cd ~/.vim/bundle/command-t/ruby
    ruby extconf.rb
    make
